# dw-apb-uart Uart driver in Rust
dw-apb-uart serial driver in Rust.

## Support boards
* T-HEAD Light Lichee Pi 4A
* BST A1000b FADA
* and more...

- _Todo: 16550 UART supports I/O port-mapped (x86 only) and memory-mapped_
